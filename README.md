## Marble.js benchmarks

**Setup**:

Just run `npm i`

**Benchmarks**:

Run benchmarks for *Marble.js*, *Koa*, *Express* and *Restify*:
```bash
npm run bench:routing      ## run routing benchmark
npm run bench:middleware   ## run middleware benchmark
```
